What is the AdiDoks?

AdiDoks is a Zola theme for Documentation's sites, ported by the Hugo Theme Doks, which is a pretty nice theme. Thanks a lot to h-enk, the creator of the Doks.

Keyboard shortcuts for search?

  • focus: /
  • select: ↓ and ↑
  • open: Enter
  • close: Esc
