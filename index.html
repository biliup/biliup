直播录制和上传工具

Biliup 是一套直播录制、视频下载和上传的工具, 提供了多种方式 自动化的完成录制上传 .

Get started

Open-source MIT License. GitHub v0.1.0

Biliup-app

B站投稿客户端，基于Tauri. 支持多p上传、线路切换、快速用户切换和多种登录方式. 核心由Biliup Rust提供支持

Fast by default ⚡️

多种上传 线路 保证各种网络环境下的满带宽上传. 支持yaml，toml配置文件格式，提供web页面快速配置，查看录制状态.

Biliup Rust

B站命令行投稿工具, 提供Flv、Hls直播流下载, 方便在服务器环境下快速投稿, 自定义自动化录制、下载、上传工具.

直播录制

检测直播. 开播自动录制，支持主流直播平台如，斗鱼、虎牙、bilibili、抖音等.

视频搬运

提供Youtube、Twitch频道搬运功能. 搬运视频列表，更新后自动上传bilibili.

自动上传

配置灵活，标题，简介支持占位符. 可配置上传后处理，如发送邮件、自动删除、备份等.
